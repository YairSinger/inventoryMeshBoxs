SPI  Shared Bus — Both PN532s

GPIOSignalConnects to
11 MOSI PN532 #1 SDI + PN532 #2 SDI
13 MISO PN532 #1 SDO + PN532 #2 SDO
12 SCK PN532 #1 SCK + PN532 #2 SCK
10 CS #1 PN532 #1 SSEL — inner reader
9 CS #2 PN532 #2 SSEL — outer reader

⚠ PN532 SPI mode: pull SEL0=LOW, SEL1=HIGH on each board before power-up. SSEL is active-low.

I2C  Master Box Only (Phase 2)

GPIOSignalConnects to
2 SDA OLED SSD1306 + MAX17043 fuel gauge
3 SCL OLED SSD1306 + MAX17043 fuel gauge

⚠ Add 4.7 kΩ pull-ups on SDA and SCL to 3.3V. OLED I2C address: 0x3C. MAX17043: 0x36.

GPIO  Lid Trigger + Buttons

GPIOSignalNotes
0 BOOT / Lid Active-low. Pulled high internally. Short to GND = lid open. Wake source for deep sleep.
4 SR04 TRIG Proto lid sensor. Output, active HIGH pulse.
5 SR04 ECHO Proto lid sensor. Input, HIGH = object detected.
14 BTN UP Master box only. Pull-up + momentary to GND.
15 BTN SELECT Master box only. Pull-up + momentary to GND.
16 BTN BACK Master box only. Pull-up + momentary to GND.

RMT  WS2812B LED

GPIOSignalNotes
48 WS2812B DIN Onboard LED. ESP-IDF RMT peripheral. 5V logic preferred — add level shifter if LED glitches at 3.3V.

NFC Reader · 13.56 MHz
MOSI: GPIO 11 · MISO: GPIO 13
SCK: GPIO 12 · CS: GPIO 9
SEL0: LOW · SEL1: HIGH (SPI mode)

NTAG213 Tags Phase 1

NFC Tags · passive

Onboard RGB LED

TRIG: GPIO 4

Master box display

Directional NFC — Turnstile Logic

Reader #1 (inner)Reader #2 (outer)WindowResult
fires first fires within 200 ms ✓ INSERT
fires within 200 ms fires first ✓ EXTRACT
Only one reader fires expires AMBIGUOUS
Second reader fires after window expired AMBIGUOUS → new pending

⚠ Reader #1 = inner (box interior side). Reader #2 = outer (box opening side). Physical placement matters for correct direction.
